## 633. 平方数之和

给定一个非负整数 `c` ，你要判断是否存在两个整数 `a` 和 `b`，使得 `a2 + b2 = c` 。

**示例 1**

```markdown
输入：c = 5
输出：true
解释：1 * 1 + 2 * 2 = 5
```

**示例 2：**

```markdown
输入：c = 3
输出：false
```

**示例 3：**

```
输入：c = 1
输出：true
```

### 代码

```java
class Solution {
    public boolean judgeSquareSum(int c) {
        Long a = 0l;
        Long b = (long)Math.sqrt(c);
        while(b >=a){
            if((a*a)+(b*b) == c){
                return true;
            }
            if((a*a)+(b*b) < c){
                a++;
            }else{   
                b--;
            }
        }
        return false;
    }
}
```
